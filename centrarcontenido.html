Centrar contenido html

Cómo centrar un texto en HTML Hay dos formas para centrar el texto en HTML, puedes usar una etiqueta HTML (obsoleta) o usar una regla CSS (método recomendado). Puedes centrar el texto con CSS mediante la propiedad text-align del elemento a centrar. Centrar un párrafo Si sólo tienes uno o unos pocos bloques de texto para centrar, añade el atributo style a la etiqueta de inicio del elemento y usa la propiedad “text-align”.
